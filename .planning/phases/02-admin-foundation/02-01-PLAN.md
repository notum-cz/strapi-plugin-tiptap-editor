  <action>
  **Step 1: Create admin/src/hooks/usePresetConfig.ts:**

  Implementation per RESEARCH.md Pattern 2 reference:
  - Import `useFetchClient` from `@strapi/strapi/admin`
  - Import `useState`, `useEffect`, `useMemo` from React
  - Import `TiptapPresetConfig`, `MINIMAL_PRESET_CONFIG` from `../../../shared/types`
  - Define `PresetConfigResult` type: `{ config: TiptapPresetConfig | null; isLoading: boolean }`
  - Export `usePresetConfig(presetName?: string): PresetConfigResult`
  - Normalize preset name: `useMemo(() => presetName?.trim(), [presetName])`
  - Initial state: if no normalizedPresetName, config = MINIMAL_PRESET_CONFIG and isLoading = false; else config = null and isLoading = true
  - useEffect with mounted guard:
    - If !normalizedPresetName: set config to MINIMAL_PRESET_CONFIG, set loading false, return
    - Async fetch: `get(`/api/tiptap-editor/presets/${normalizedPresetName}`)`
    - On success: `setConfig(response.data || MINIMAL_PRESET_CONFIG)`
    - On catch: `console.warn('[TiptapEditor] Failed to fetch preset config:', error)` then `setConfig(MINIMAL_PRESET_CONFIG)`
    - Finally: `setIsLoading(false)` (only if mounted)
    - Cleanup: `return () => { mounted = false }`
  - Dependencies: `[get, normalizedPresetName]`
